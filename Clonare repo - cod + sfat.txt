Ca sa clonati repo-ul local, folositi:
git clone https://github.com/Raimond-Eduard/IP-proiect-2024.git

Daca nu apare ca ati avea drepturi folositi:

git user.name "Numele vostru"

git user.email "emailul pe care il folositi pe git si este adaugat in proiect"

Dupa asta incercati sa dati clone din nou
